Why Pyramid is awesome: the example of Facebook applications
PyCon US 2013
Éric Araujo
Recorded: March 17, 2013 — Language: English

Beside the friends, walls and photos, Facebook is a platform where anybody can build applications or games taking advantage of the huge user base and connections between people. Any web-capable language or framework can be used to do that, but this poster will show you why Pyramid is a most excellent choice.

How you can contribute to Python
PyCon CA 2012
Éric Araujo
Recorded: Nov. 11, 2012 — Language: English

Did you know that you could contribute to Python? From documentation

work to patch reviews to new features to blogging, there are a number of

ways you can give back to Python. This talk for beginners will walk you

through these ways with concrete demonstrations.
