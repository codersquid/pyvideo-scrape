This poses a great challenge to researchers who need exactly the same numerical results from run to run, and across different systems.\n\nThis talk describes how to use tools such as Intel\u00ae Math Kernel Library (Intel\u00ae MKL) and Intel\u00ae compilers to build numerical reproducibility into Python based tools. Intel\u00ae compilers provide broader solutions to ensure the compiler-generated code produces reproducible results. Performance impact of each level of controls is discussed in detail. Our conclusion is that, there is usually a certain degree of trade-off between reproducibility and performance.
